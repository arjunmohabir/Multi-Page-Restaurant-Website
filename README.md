# Multi-Page Restaurant Website

## Overview
This project is a multi-page restaurant website built using HTML. It showcases a Caribbean-style menu with multiple pages, images, and structured navigation between sections.

## Objective
The goal of this project was to design and organize a static website with multiple pages, demonstrating understanding of HTML structure, navigation, and content layout.

## Features
- Multi-page navigation (Home, Menu, Additional Services)
- Image-based menu and food presentation
- Structured layout using HTML
- Internal linking between pages

## Pages Included
- `index.html` → Homepage
- `menu.html` → Food menu with images
- `otherservices.html` → Additional offerings/services

## Project Structure
```
project/
├── index.html
├── menu.html
├── otherservices.html
├── images (food + background assets)
```

## Key Concepts
- HTML page structure
- Internal linking and navigation
- Organizing content across multiple pages
- Basic UI layout for a restaurant website
